---

## ユーザーペルソナ

### ペルソナ1: 来店顧客

| 項目 | 内容 |
|------|------|
| 名前 | 山田 花子（35歳） |
| 属性 | 会社員、ランチタイムによく外食する |
| 目標 | 限られた昼休みを有効活用しながら、人気店で食事したい |
| 課題 | 行列に並ぶ時間がもったいない、順番を逃すのが不安 |

### ペルソナ2: 店舗スタッフ

| 項目 | 内容 |
|------|------|
| 名前 | 佐藤 太郎（28歳） |
| 属性 | 飲食チェーン店長、ランチタイムは常に混雑 |
| 目標 | 混雑時でも効率的に顧客を案内し、満足度を維持したい |
| 課題 | 行列整理に人手を取られ、接客品質が落ちる |

## ユーザーアクティビティ（大きな流れ）

### 来店顧客の流れ

```
[来店・整理券取得] → [待機・順番確認] → [呼び出し・入店] → [会計・退店]
```

### 店舗スタッフの流れ

```
[顧客受付] → [順番管理] → [顧客呼び出し] → [来店データ確認]
```

## ストーリーマップ

### アクティビティ1: 来店・整理券取得

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| QRコードスキャン | As a 来店顧客, I want to 店頭のQRコードをスキャンする, So that 簡単に整理券を取得できる | MVP |
| 整理券発行 | As a 来店顧客, I want to 整理券番号と待ち組数を確認する, So that 自分の順番がわかる | MVP |
| LINE友だち追加 | As a 来店顧客, I want to LINEで友だち追加する, So that 通知を受け取れる | MVP |
| 人数入力 | As a 来店顧客, I want to 来店人数を入力する, So that 適切な席に案内される | 後回し |

### アクティビティ2: 待機・順番確認

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| リアルタイム確認 | As a 来店顧客, I want to あと何組かをリアルタイムで確認する, So that 戻るタイミングがわかる | MVP |
| 順番通知受信 | As a 来店顧客, I want to 順番が近づいたらLINE通知を受け取る, So that 順番を逃さない | MVP |
| 会員登録 | As a 来店顧客, I want to 待ち時間中に会員登録する, So that 次回以降お得に利用できる | MVP |
| 待ち時間予測 | As a 来店顧客, I want to 予想待ち時間を確認する, So that 時間の使い方を決められる | 後回し |

### アクティビティ3: 呼び出し・入店

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| 呼び出し通知 | As a 来店顧客, I want to 順番が来たらLINE通知を受け取る, So that すぐに入店できる | MVP |
| 入店確認 | As a 店舗スタッフ, I want to 顧客の入店を確認する, So that 順番を進められる | MVP |
| スキップ処理 | As a 店舗スタッフ, I want to 不在の顧客をスキップする, So that 他の顧客を待たせない | 後回し |

### アクティビティ4: 来店データ活用

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| 来店履歴確認 | As a 店舗スタッフ, I want to 顧客の来店履歴を確認する, So that リピーターを把握できる | 後回し |
| セグメント配信 | As a 店舗スタッフ, I want to 来店後に自動でメッセージを送る, So that リピートを促進できる | 後回し |

## MVPスコープ

### MVP機能一覧

1. **QRコード整理券発行**: 店頭QRスキャンで整理券取得
2. **リアルタイム順番確認**: あと何組かの表示
3. **LINE通知（順番接近・呼び出し）**: プッシュ通知配信
4. **会員登録導線**: 待機画面からの会員登録フォーム
5. **入店確認機能**: スタッフ側の顧客入店処理

### 後回し機能

1. 人数・席種指定
2. 待ち時間予測表示
3. 顧客スキップ・再呼び出し
4. 来店履歴・セグメント配信連携
5. 発券機連携
