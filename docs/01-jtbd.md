# JTBD (Jobs To Be Done)

## プロダクト名: LINE順番待ち

## メインジョブ

When **店舗が混雑していて順番待ちが発生している時**
I want to **スマホから整理券を取得し、自由に時間を使いながら順番を把握したい**
So that **待ち時間のストレスなく、自分のペースで来店できる**

## 関連ジョブ

### ジョブ1: 店舗スタッフの業務効率化

When **ランチタイムなど店舗が混雑している時**
I want to **行列整理や顧客呼び出しを自動化したい**
So that **接客など本来の業務に集中できる**

### ジョブ2: 顧客の会員化促進

When **順番待ち中の顧客がいる時**
I want to **自然な流れでLINE友だち追加・会員登録を促したい**
So that **将来のリピート集客につなげられる**

### ジョブ3: 感染症対策・密回避

When **待合室に多くの顧客が滞留している時**
I want to **院外・店外で待機してもらいたい**
So that **3密を回避し、安心・安全な環境を提供できる**

### ジョブ4: 来店データの収集

When **顧客が順番待ちで来店した時**
I want to **顧客属性やリピート状況を把握したい**
So that **セグメント配信など個別のマーケティングに活用できる**

## ジョブの優先順位

| 優先度 | ジョブ | 理由 |
|--------|--------|------|
| 1 | 顧客の待ち時間ストレス解消 | 顧客体験の向上が最も直接的な価値提供 |
| 2 | 店舗スタッフの業務効率化 | 人件費削減・サービス品質向上に直結 |
| 3 | 顧客の会員化促進 | 待ち時間を活用した自然な会員獲得が差別化ポイント |
| 4 | 来店データの収集・活用 | 長期的なCRM施策の基盤となる |
